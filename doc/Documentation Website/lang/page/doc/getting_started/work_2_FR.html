PHosPhore intègre un système de permission. Chaque visiteur possède un nom de rôle ("role_name"), qui possède des permissions d'accès à certains couples application/action. Tout cela est géré dans la base de donnée: la table "permission" regroupe les permissions, liant un "role_name" à un couple application/action. Attention, même la page d'erreur doit être dans les permissions de chaque rôle pour que chaque rôle puisse y accéder !
