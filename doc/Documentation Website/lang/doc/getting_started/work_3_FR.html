Dans PHosPhore, chaque chargement de page est effectué par un "Visiteur" (class /user/Visitor). A l'arrivé sur le site, tout les utilisateurs sont considérés comme visiteur "guest". Si vous ne créez pas de système d'inscription vous-même, chaque visiteur sera toujours guest. C'est l'existence de cet objet (stocké dans la variable $Visitor) qui permet le système de permission. Entre chaque page, le pseudo, l'id et le mot de passe EN CLAIR seront stocké dans la session, et l'instance de /user/Visitor sera recréé, en vérifiant la correspondance mot de passe/pseudo. Cela permet aussi de créer une configuration spéciale pour chaque utilisateur (par défaut, la modification de la configuration ne fonctionne pas pour le visiteur "guest")
