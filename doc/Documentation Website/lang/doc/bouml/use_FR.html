BOUML est téléchargeable sur son site officiel. Une aide pour l'installation est disponible sur le site officiel.
Vous pouvez ensuite lancer le logiciel, et créer un nouveau projet.

Le dossier du projet peut être là où vous le souhaitez. Ensuite, une alerte devrait apparaître. Validez et définissez PHP comme étant le langage de ce projet.

Vous pouvez maintenant réverser votre code PHP, en sélectionnant le dossier contenant tout votre code (contenant le index.php)

Il ne reste qu'à générer la documentation HTML, dans le dossier de votre choix

Vous pouvez accéder à cette documentation de la même façon qu'avec celle de PHosPhore
