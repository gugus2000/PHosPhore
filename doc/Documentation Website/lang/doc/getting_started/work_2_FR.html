PHosPhore intègre un système de permission. Chaque visiteur possède un "role_name", qui possède des permissions d'accès à certains couple application/action. Tout cela est géré dans la base de donnée: la table "permission" regroupe les permissions, liant un "role_name" à un couple application/action. Attention, même la page d'erreur doit être dans les permissions de chaque "role_name" pour que chaque "role_name" puisse y accéder !
