Pour utiliser du contenu dynamique, deux "types" de fichiers peuvent êtres utilisés: les fichiers lang et config. Ces deux types de fichiers sont présent dans des dossiers distincts. Les fichiers à la racine de ces dossiers sont toujours chargés. Puis, les fichiers concernant l'application cible sont chargés, puis ceux de l'action. La structure des dossiers reprends donc celle du dossier "page".
