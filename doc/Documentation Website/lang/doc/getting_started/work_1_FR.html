Dans PHosPhore, chaque page est une "action" contenu dans une "application". Une même application peut contenir plusieurs actions, à l'image d'un dossier contenant plusieurs fichiers. La position des fichiers du site dépendent de ce couple. Ainsi, la définition de la page se fait toujours dans page/application/action/page.php. Dans les paramètre, il est nécessaire de définir l'application par défaut si ni l'application, ni l'action est chargé, et de définir, pour chaque application, l'action par défaut si aucune n'est précisé. Pour une application "a" et une action "b":

  • Définir default_application à "a" va permettre que l'application utilisée par défaut soit "a".
  • Définir a_default_action à "b" va permettre qu'une utilisation de l'application "a" sans précision d'action va utiliser l'action "b".
    • Ces deux paramètres sont nécessaires. Et ce, pour le dernier, pour chaque application existante.
