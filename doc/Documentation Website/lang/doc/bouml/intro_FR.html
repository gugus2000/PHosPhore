BOUML est un logiciel permettant de générer du code à partir de diagrammes UML ou inversement.
À chaque nouvelle version de PHosPhore, les diagrammes UML des classes du framework sont regénéré avec BOUML.
L'outil dispose aussi d'un système de génération de documentation HTML à partir des diagrammes, qui est, de même, utilisé à chaque nouvelle version de PHosPhore.
C'est cette documentation HTML généré par BOUML qui va être le sujet de cette page.
