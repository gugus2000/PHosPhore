La première configuration du serveur se fait en éditant le fichier "config.php" dans le dossier "config". La configuration est représentée par une array. Seuls quelques paramètres ont besoin d'être modifiés si vous souhaitez conserver le fonctionnement par défaut du framework:

  • db_host doit contenir le nom de l'hôte utilisé par le framework.
  • db_name doit contenir le nom de la base de donnée dans laquelle vous avez importé "example.sql".
  • db_username doit contenir un nom d'utilisateur ayant accès à la base de donnée db_name.
  • db_passwd doit contenir le mot de passe de db_username.
  • general_name doit contenir le nom de votre site.
  • general_email doit contenir votre adresse mail pour le site.
Une fois ces valeurs rentrés, vous pouvez vérifier le bon fonctionnement de PHosPhore en chargeant le site. Si tout fonctionne, vous devez voir apparaître "It works!" dans votre navigateur. Sinon, le framework est mal configuré: vérifiez vos valeurs entrés dans le fichier de configuration.
