Le téléchargement de PHosPhore s'effectue ici (d'autres miroirs de téléchargements seront disponible par la suite). Arrivé sur le site, il suffit de cliquer sur "Clone or download" (en vert), puis "Download ZIP". Vous pouvez ensuite extraire le fichier (avec 7-ZIP par exemple). Dans le dossier extrait du fichier ZIP, vous trouverez un dossier nommé "main". C'est ce dossier qui sera utilisé par votre site web.
