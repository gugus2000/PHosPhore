Une classe héritant de /core/Managed doit contenir certaines méthodes particulières pour chaque attribut existant. Pour un attribut "abcd" du type "type", il doit exister:

  • La méthode public : getAbcd( void ) : type qui renvoie l'attribut.
  • La méthode protected : setAbcd( type : $abcd ) : void qui permet de définir l'attribut (re-définition du type conseillé).
  • La méthode public : displayAbcd( void ) : string qui permet l'affichage sécurisé de l'attribut en un string lisible pour l'être humain.
