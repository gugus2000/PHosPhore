Une classe héritant de \core\Managed doit contenir certaines méthodes particulières pour chaque attribut existant. Pour un attribut "abcd" du type "type", il doit exister:

  • La méthode public : getAbcd( void ) : type qui renvoie la valeur de "abcd".
  • La méthode protected : setAbcd( type : $abcd ) : void qui permet de définir "abcd".
  • La méthode public : displayAbcd( void ) : string qui permet l'affichage sécurisé de "abcd" en un string lisible pour l'être humain (n'oubliez pas d'utiliser htmlspecialchars( string ) ).
